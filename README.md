# When a Valley Disappeared — Hetch Hetchy Digital Exhibit

A multi-page digital exhibit analyzing Gifford Pinchot's 1913 congressional testimony in support of the Raker Act.

---

## HOW TO GET YOUR LIVE LINK — Step by Step

### Step 1: Create a GitHub Account
1. Go to **github.com**
2. Click **Sign up** (top right)
3. Choose a username (this becomes part of your URL — e.g. `janedoe`)
4. Enter your email and a password
5. Verify your email when they send you a confirmation

---

### Step 2: Create a New Repository
1. After logging in, click the **+** button (top right corner) → **New repository**
2. Name it exactly: `hetch-hetchy-exhibit`
3. Make sure it is set to **Public** (required for free GitHub Pages)
4. Check the box that says **Add a README file**
5. Click **Create repository**

---

### Step 3: Upload the Website Files
1. Inside your new repository, click **Add file** → **Upload files**
2. Drag and drop ALL of the following files at once:
   - `index.html`
   - `context.html`
   - `figures.html`
   - `stakes.html`
   - `testimony.html`
   - `outcome.html`
   - `sources.html`
   - `style.css`
   - `nav.js`
3. Scroll down and click **Commit changes**

---

### Step 4: Turn on GitHub Pages
1. Go to your repository's **Settings** tab (top menu bar)
2. In the left sidebar, click **Pages**
3. Under **Source**, click the dropdown that says "None" and select **main**
4. Make sure the folder says **/ (root)**
5. Click **Save**

---

### Step 5: Get Your Live Link
- GitHub will show a green box saying **"Your site is published at..."**
- Your URL will be: `https://YOUR-USERNAME.github.io/hetch-hetchy-exhibit/`
- **It may take 1–5 minutes** for the site to go live — refresh the page if it doesn't appear immediately

---

## Your Final URL Format
```
https://YOUR-USERNAME.github.io/hetch-hetchy-exhibit/
```
Replace `YOUR-USERNAME` with whatever username you chose in Step 1.
